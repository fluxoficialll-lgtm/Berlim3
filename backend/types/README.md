# Tipos (Types)

Este diretório contém as definições de tipo do TypeScript para a aplicação. O uso do TypeScript ajuda a garantir a segurança de tipo e a clareza do código, especialmente em projetos maiores.

## Propósito

- **Definir Estruturas de Dados**: Definir interfaces e tipos para os objetos de dados que são usados em toda a aplicação (por exemplo, `User`, `Post`, `Group`).
- **Melhorar a Experiência do Desenvolvedor**: Fornecer autocompletar e verificação de tipo em tempo de desenvolvimento, o que ajuda a evitar erros e a entender a estrutura do código.
- **Documentar o Código**: As definições de tipo servem como uma forma de documentação, descrevendo a forma dos dados que são esperados por diferentes partes da aplicação.

## Exemplos

- **`user.types.ts`**: Contém as definições de tipo para os objetos de usuário.
- **`social.types.ts`**: Pode conter tipos relacionados a interações sociais, como comentários, curtidas, etc.
- **`financial.types.ts`**: Definições de tipo para transações financeiras, faturas, etc.
