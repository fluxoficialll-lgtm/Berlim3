# Backend

Este diretório contém o código do backend da aplicação, construído com Node.js e Express.

## Começando

### Pré-requisitos

- Node.js (versão 20 ou superior recomendada)
- npm

### Instalação

1.  Navegue para o diretório `backend`:
    ```bash
    cd backend
    ```
2.  Instale as dependências:
    ```bash
    npm install
    ```

### Executando o Servidor

Para iniciar o servidor, execute:

```bash
npm start
```

Isso iniciará o servidor Express, e ele estará escutando por requisições da API.

## Dependências Chave

- **Express**: Framework web para Node.js.
- **Mongoose**: Biblioteca de Modelagem de Dados de Objeto (ODM) para MongoDB.
- **Socket.IO**: Para comunicação bidirecional em tempo real.
- **Firebase Admin**: Para interagir com os serviços do Firebase a partir do backend.
