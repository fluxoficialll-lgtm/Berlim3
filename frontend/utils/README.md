# Utilitários (Utils)

Este diretório contém funções auxiliares e utilitários que são usados em toda a aplicação. O objetivo é ter um local central para funções genéricas e reutilizáveis que não se encaixam em nenhuma outra categoria específica.

## Propósito

Os utilitários ajudam a manter o código limpo, evitando a duplicação de lógica comum. Eles devem ser funções puras sempre que possível, o que significa que, para a mesma entrada, eles sempre produzirão a mesma saída, sem efeitos colaterais.

## Exemplos

- **`idGenerator.ts`**: Uma função para gerar IDs únicos.
- **Funções de formatação de data**: Para exibir datas em formatos legíveis.
- **Funções de validação**: Para verificar a validade de entradas de usuário, como e-mails ou senhas.
